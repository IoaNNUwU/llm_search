# Сборка мусора

**Мусор** - неиспользуемая память.

## Виды сборки мусора

`Go` **Трассирование** (Трейсинг) - Вычисление графа используемых объектов начиная со стека и глобальных переменных.

`not Go` **Подсчёт ссылок** - изменение количества ссылок при изменении счётчика. Проблема - производительность и циклические ссылки. Плюсы - отсутствие отдельного процесса, синхронизация, отсутствие пауз на сборку мусора.

## Mark and Sweep (трёхцветный алгоритм разметки)

**Mark** - пометка в 3 цвета с целью поиска цикла. Если исследуемый **серый** объект указывает на уже исследованный, то это цикл.

- **Чёрный** - исследованный.

- **Серый** - ожидает исследования.

- **Белый** - неисследованный.

### Sweep

`Go` **Sweep** - удаление мёртвых объектов.

`not Go` **Copy** - перенести живые на новое место.

В конце удаляются белые - оставшиеся после обхода графа

## Чем больше мусора, тем больше пауза

Сборщик мусора [Конкуретный](../pattern/concurrency.md#concurrent).

### При параллельном доступе:

- если **чёрный** объект начинает ссылаться на новый, то автоматически **красим в серый**.

- если **объект удаляется**, то он перекрашивается в серый.

Для включения и отключения барьера нужен `Stop the World`.

- В `Go` **нет поколений объектов** как в других языках, благодаря большей роли аллокаций на стеке.

## Когда запускается сборка мусора

1. Превышение **лимита кучи**, установленного с помощью переменной `GOGC`. То есть если после очистки мусора размер кучи `2GB`, то при `GOGC = 100%` следующая сборка будет запущена на `4GB`.

2. Каждые **2 минуты**

3. Ручной запуск **runtime.GC()**

## Death Spiral

```txt
Нужно больше памяти -> Чаще GC 
    -> Тратим больше CPU на GC -> скорость падает
        -> запросы копятся в очереди
```

## Переменные

Переменная `GOMEMLIMIT=2750MB` - рантайм контролирует ресурсы сборщика мусора и не даёт ему весь CPU. При этом можно аллоцировать выше мем лимита, это всего лишь мягкое таргет значение.

Переменной `GOGC=100%`- после очистки мусора размер кучи `2GB`, то при `GOGC = 100%` следующая сборка будет запущена на `4GB`.
